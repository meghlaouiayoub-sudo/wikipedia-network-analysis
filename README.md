# Wikipedia Network Analysis
This is a study on the musae_chameleon Wikipedia network using Python and NetworkX.

## Dataset
The dataset is a network of Wikipedia pages.
- Nodes: Wikipedia pages
- Edges: Hyperlinks between pages

Files in the Dataset:
- musae_chameleon_edges.csv
- musae_chameleon_features.json
- musae_chameleon_target.csv

## Network Statistics
Nodes: 2,277  
Edges: 31,421  
Density: 0.0121  
Average clustering coefficient: 0.4813  
Average path length: 3.56  

## Analysis
The analysis includes:
- Degree distribution
- Connected components
- Centrality measures
- Clustering coefficient
- Average path length
- Comparison to standard network models: Erdős–Rényi model, Barabási–Albert model, Watts–Strogatz model

## Tools Used
- Python
- NetworkX
- Pandas
- Matplotlib
- Jupyter Notebook
